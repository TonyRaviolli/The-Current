# UnderCurrent — Agent Instructions

Brand:
- Premium intelligence briefing. Dark abyss theme, gold/teal accents.
- Typography: serif for headlines, sans for body, mono for metadata.

UI Principles:
- Clear hierarchy, generous whitespace, consistent spacing scale.
- Avoid template vibes. Subtle depth, restrained motion.

Engineering:
- Prefer small, readable JS modules over monolithic scripts.
- Data-driven rendering; avoid repeated HTML blocks.
- Accessibility: focus states, keyboard interaction, Escape closes modals/menus.

Don’t:
- Add frameworks unless explicitly requested.
- Don’t change :root color tokens unless necessary.
